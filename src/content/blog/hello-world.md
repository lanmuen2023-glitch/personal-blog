## Welcome

This is my first blog post. The blog is built with **Astro** and **Tailwind CSS v4**, deployed on **Vercel**.

## Why Astro?

Astro is a modern static site generator that ships **zero JavaScript by default**. Perfect for content-focused websites like blogs.

### Code Block Example

```js
const greeting = "Hello, world!";
console.log(greeting);
```

### Features

- Markdown for writing
- Tags and categories
- Dark mode support
- RSS feed
- SEO optimized
